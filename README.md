# 📊 Code Coverage Dashboard

A centralized dashboard that aggregates code coverage reports from multiple repositories into a single, easy-to-use interface.

Instead of checking coverage reports individually across repositories, this dashboard provides a unified view of code coverage metrics, enabling teams to monitor project health, identify gaps in testing, and track overall code quality.

---

## ✨ Features

- 📈 View code coverage from multiple repositories in one dashboard
- 📂 Automatically parses Cobertura XML coverage reports
- 🔍 Repository-wise coverage statistics
- 📊 Interactive and responsive dashboard
- 🔐 Secure Google SSO Authentication
- ⚡ Fast REST APIs built with FastAPI
- 🎨 Modern React-based frontend
- ☁️ Frontend deployed on Vercel
- 🖥️ Backend hosted on AWS EC2

## 🚀 Getting Started

```bash

```

## Frontend Setup

```bash

npm start
```

The frontend will start on

```

```

## Backend Setup

Create a virtual environment

```bash
python -m venv venv
```

Activate it

### Windows

```bash
venv\Scripts\activate
```

### macOS/Linux

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the server

```bash
uvicorn main:app --reload
```

Backend runs on

```
http://localhost:8000
```

## 📈 How It Works

1. Coverage reports are generated by different repositories in **Cobertura XML** format.
2. The FastAPI backend parses these XML reports.
3. Coverage statistics are extracted and exposed through REST APIs.
4. The React frontend fetches this data.
5. Users can securely log in using Google SSO to view the dashboard.
6. The dashboard presents repository-wise coverage metrics in a clean and centralized interface.

Cobertura is a widely supported XML format for reporting test coverage, making it suitable for aggregating coverage across multiple projects. :contentReference[oaicite:0]{index=0}
